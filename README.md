# Social Network Analysis Tool

This project is a tool for analyzing social networks using various graph algorithms.
## Used Technologies: - 
Programming Language: Python
Libraries: NetworkX, Matplotlib

## Features

- **BFS Traversal**
- **DFS Traversal**
- **Dijkstra's Shortest Path**
- **Centrality Measures**
- **Graph Visualization**
