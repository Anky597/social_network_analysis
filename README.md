This project is a tool for analyzing social networks using various graph algorithms.
## Used Technologies: - 
Programming Language: Python
Libraries: NetworkX, Matplotlib, Plotly , Sickit Learn , Pandas, Numpy.

## Features

- **BFS Traversal**
- **DFS Traversal**
- **Dijkstra's Shortest Path**
- **Graph Visualization**
- **Clusering Algorithm**
